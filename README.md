# PriceTracker 📊

个人日用品价格追踪与性价比分析工具

## 功能特性

- 📋 **价格总览** - 所有物品的汇总信息，支持搜索、筛选、排序
- 🛒 **购买记录** - 每次购买的详细流水，支持多条件筛选
- 📈 **数据分析** - 可视化图表：分类占比、渠道占比、月度趋势、日均成本排行
- 🤖 **AI 智能导入** - 拍照识别小票，AI 自动解析为结构化数据
- 📥📤 **导入导出** - 支持 JSON/CSV 格式
- 🌙 **暗色模式** - 自动跟随系统主题
- 📱 **响应式设计** - 桌面、平板、手机均可使用

## 使用方法

### 方式一：直接打开

双击 `open-app.bat`，或用浏览器打开 `index.html`

### 方式二：本地服务器

双击 `start-server.bat`，或运行：

```bash
python -m http.server 8080
```

然后访问 http://localhost:8080

## 核心计算逻辑

### 平均使用时长

相邻两次购买同一物品的时间差 ÷ 上次购买的数量

```
2026-01-01  抽纸  qty=10
2026-01-31  抽纸  qty=10   →  30天 ÷ 10 = 3天/包
2026-03-02  抽纸  qty=5    →  31天 ÷ 10 = 3.1天/包
平均 = (3 + 3.1) / 2 ≈ 3天/包
```

特殊处理：
- 间隔 ≤ 3 天视为囤货，不计入
- 只有 1 次记录显示「数据不足」
- 最后一次购买用今天估算

### 日均成本

```
日均成本 = 平均单价 ÷ 平均使用时长
```

## AI 智能导入

1. 点击「复制 AI 提示词模板」
2. 将模板和小票照片一起发给 ChatGPT / Claude / 豆包等 AI
3. AI 返回 JSON 后粘贴导入

支持拍照识别、文字描述、批量导入

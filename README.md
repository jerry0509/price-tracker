# PriceTracker 📊

个人日用品价格追踪与性价比分析工具

## 功能特性

- 📋 **价格总览** - 所有物品汇总，支持搜索、筛选、排序，显示归一化单价
- 🛒 **购买记录** - 每次购买的详细流水，支持多条件筛选和分页
- 📈 **数据分析** - 可视化图表：分类占比、渠道占比、月度趋势、日均成本排行
- 🏷️ **促销标记** - 记录促销商品，自动计算实付价，统计时按实付价计算
- 📏 **规格追踪** - 支持不同规格（120抽/250ml/30枚），自动计算单位价格
- 🤖 **AI 智能导入** - 拍照识别小票，AI 自动解析为结构化数据
- 📥📤 **导入导出** - 支持 JSON/CSV 格式，CSV 包含全部字段
- 🌙 **暗色模式** - 自动跟随系统主题
- 📱 **响应式设计** - 桌面、平板、手机均可使用

## 使用方法

### 方式一：直接打开

双击 `open-app.bat`，或用浏览器打开 `index.html`

### 方式二：本地服务器

双击 `start-server.bat`，或运行：

```bash
python -m http.server 8080
```

然后访问 http://localhost:8080

## 核心功能

### 规格追踪

同一种商品可能有不同规格（牛奶 250ml vs 1L，鸡蛋 10枚 vs 30枚），系统支持记录每笔购买的规格信息，自动计算归一化单价：

```
归一化单价 = 实付金额 ÷ (规格数量 × 购买数量)
```

### 促销标记

勾选「促销」后可记录促销方式和实付金额，统计时自动按实付价计算。

支持的促销类型：买一送一、第二件半价、限时特价、满减、优惠券、折扣等。

### 平均使用时长

相邻两次购买同一物品的时间差 ÷ 上次购买的数量

```
2026-01-01  抽纸  qty=10
2026-01-31  抽纸  qty=10   →  30天 ÷ 10 = 3天/包
2026-03-02  抽纸  qty=5    →  31天 ÷ 10 = 3.1天/包
平均 = (3 + 3.1) / 2 ≈ 3天/包
```

特殊处理：
- 间隔 ≤ 3 天视为囤货，不计入
- 只有 1 次记录显示「数据不足」
- 最后一次购买用今天估算

### 日均成本

```
日均成本 = 平均单价 ÷ 平均使用时长
```

### AI 智能导入

1. 点击「复制 AI 提示词模板」
2. 将模板和小票照片一起发给 ChatGPT / Claude / 豆包等 AI
3. AI 返回 JSON 后粘贴导入

支持识别规格信息和促销信息。

# 气质定位

## 现状诊断

当前设计存在的"AI味"问题：
- ❌ 背景网格+光晕效果（`bg-grid`, `bg-glow`）→ 过度装饰
- ❌ 副标题"智能消费分析 · AI 驱动"→ 假大空文案
- ❌ 过多渐变（6个gradient变量）→ 视觉疲劳
- ❌ 圆角过大（`radius-xl: 24px`）→ 幼稚感
- ❌ emoji当功能性图标（📊🛒📈🤖）→ 不专业

## 气质定位

### 参考对象
**Notion 的数据库视图 + Apple 的设置页面**

- **Notion 数据库视图**：清爽的表格布局、克制的配色、信息密度高但不杂乱
- **Apple 设置页面**：清晰的分组、统一的列表样式、直觉的操作逻辑

### 整体感受
**冷静 · 专业 · 克制**

- 像一个认真记账的日本人：数据精确、界面干净、没有花哨的东西
- 不是"智能AI助手"，而是"可靠的记账本"
- 用数据说话，不用装饰说话

### 核心视觉记忆点
**深色背景上的清晰数据卡片**

- 深色背景（`#0d1117`）提供沉浸感
- 数据卡片用微妙的边框和阴影分层
- 价格数字用等宽字体，强调"精确感"
- 唯一的强调色用于关键操作（添加记录、价格变化）

## 设计原则

### 1. 去装饰化
- 移除背景网格和光晕
- 移除无意义的渐变
- 圆角控制在 8-12px
- 只保留必要的阴影

### 2. 信息优先
- 数据是主角，UI是配角
- 表格行高适中，信息密度合理
- 数字用等宽字体，便于对比
- 颜色仅用于语义（成功/警告/危险）

### 3. 一致的视觉语言
- 按钮：ghost风格（无背景）为主，primary仅用于主操作
- 图标：统一使用SVG图标，不用emoji
- 间距：基于8px网格系统
- 字体：系统字体栈，数字用等宽字体

### 4. 克制的交互反馈
- hover：微妙的背景色变化（0.15s）
- 点击：轻微的缩放（0.98）
- 状态变化：淡入淡出（0.3s）
- 无弹跳、无缩放、无旋转动画

## 与现有设计的对比

| 元素 | 现有设计 | 新定位 |
|------|----------|--------|
| 背景 | 网格+光晕 | 纯色 `#0d1117` |
| 副标题 | "智能消费分析 · AI 驱动" | 移除 |
| 圆角 | 24px | 8-12px |
| 渐变 | 6个 | 0个（仅保留必要的状态色） |
| 图标 | emoji | SVG |
| 强调色 | 紫色 `#6c5ce7` | 绿色 `#238636`（GitHub风格） |
| 字体 | 系统字体 | 系统字体 + 等宽数字 |
